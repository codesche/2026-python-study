#### 🔹 딕셔너리 기본 패턴

##### 📌 값 증가 기본 패턴 (가장 중요)
```python
counter = {}

items = ["a", "b", "a", "c", "b", "a"]

for item in items:
    if item not in counter:
        counter[item] = 1
    else:
        counter[item] += 1

print(counter)
```

### ✅ Day 09 – 재귀(Recursion) & DFS 기초
📅 학습일: 2026-01-10

#### 🔹 학습한 내용
- 재귀(Recursion)의 개념과 구조 이해
- 종료 조건(Base Case)의 중요성 인식
- 팩토리얼 및 문자열 뒤집기 재귀 구현

- 재귀 호출은 스택 구조로 동작
- DFS는 재귀와 가장 자연스럽게 결합
- visited 처리를 하지 않으면 무한 루프 발생
- 격자 DFS는 방향 배열과 함께 자주 사용
